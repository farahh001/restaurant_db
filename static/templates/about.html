About Us

We are all computer science students at The City College of New York. This project was a collaboration amongst the four of us for our CSC 33600 class (Intro to Database Systems). The purpose of this project was to understand how to design a database given a dataset, and then creating a fullstack application for ease of access of the database.

This project implements the use of various technologies including, MySQL, Flask, Xammp. The following languages were used to build this application: Python, SQL, javascript, html, and css.
